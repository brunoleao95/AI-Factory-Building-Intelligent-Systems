# Documentos para RAG

Coloque aqui os documentos tecnicos que deseja indexar para busca semantica.

## Formatos aceitos
- `.txt` (texto puro)
- `.pdf` (documentos PDF)

## Exemplos de documentos uteis
- Manual DSM-5 (resumos)
- Tecnicas de TCC (Terapia Cognitivo-Comportamental)
- Codigo de Etica do Psicologo (CFP)
- Artigos cientificos relevantes

## Como funciona
1. Coloque os arquivos nesta pasta
2. O sistema indexa automaticamente ao iniciar
3. Faca perguntas no chat e o assistente buscara nos documentos
