# Entregas do Projeto AI Factory

Este diretorio contem o registro tecnico de cada etapa do projeto da disciplina **AI Factory: Building Intelligent Systems** (PUCPR, 2o ano).

A pasta esta separada de `docs/` propositadamente: `docs/` e o corpus indexado pelo RAG do chatbot, e meta-documentacao misturada com aquele conteudo poluiria as respostas tecnicas para a psicologa.

## Status das Entregas

| Etapa | Documento | Status | Foco |
|-------|-----------|--------|------|
| 1 | [etapa1.md](etapa1.md) | Entregue | Construcao da base: chat, RAG, Text-to-SQL, dados sinteticos |
| 2 | [etapa2.md](etapa2.md) | Entregue | Camadas de inteligencia: AutoML, agentes CrewAI, observabilidade Langfuse, testes DeepEval |
| 3 | [etapa3.md](etapa3.md) | Em andamento | Agir: seguranca (guardrails + PII), deploy hibrido, analise critica |

## Como ler

- **etapa1.md** — escrita retrospectivamente para preservar o estado do projeto antes da etapa 2.
- **etapa2.md** — relatorio tecnico exigido pela rubrica (700-1500 palavras), base do PDF entregue ao professor.
- **etapa3.md** — documento de analise critica (Partes A/B/C): revisao do desafio, analise tecnica e visao de futuro.
- **screenshots/** — capturas de tela referenciadas nos relatorios.
